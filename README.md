# bailey99beats

Personal beats library and player app with iOS glassmorphism design.

Upload, organize, and play your beats in a sleek Apple Music inspired interface.

## Features

- Upload audio files directly from your device
- Tag beats with title, category (Trap, Afrobeat, Lo-Fi, Drill, R&B, Hip Hop), and BPM
- Auto generated gradient cover art for each beat
- Filter library by category
- Full playback controls: play/pause, skip, seek
- Expandable Now Playing view with waveform visualizer
- PWA ready: install on your phone home screen
- Works offline once loaded

## How to Use

### Option 1: Open directly

Just open `index.html` in any browser. That's it.

### Option 2: Host on GitHub Pages (recommended)

1. Create a new repository on GitHub named `bailey99beats`
2. Push this project:

```bash
git init
git add .
git commit -m "initial commit"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/bailey99beats.git
git push -u origin main
```

3. Go to your repo **Settings > Pages**
4. Under "Source", select **main** branch and **/ (root)**
5. Click Save
6. Your app will be live at `https://YOUR_USERNAME.github.io/bailey99beats/`

### Option 3: Install as an app on iPhone

1. Open your GitHub Pages URL in Safari
2. Tap Share > Add to Home Screen
3. Name it "bailey99beats" and tap Add

## Tech Stack

- React 18 (via CDN)
- Pure CSS animations and glassmorphism
- Web Audio API for playback
- PWA manifest for home screen install
- Zero build step, zero dependencies to install
